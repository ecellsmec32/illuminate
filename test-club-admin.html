Features to Test

  • Authentication: Dashboard should redirect to login if not authenticated
  • Role Check: Only club organizers should access the dashboard
  • Data Loading: Dashboard should load club-specific data
  • Dark Mode: Toggle between light and dark themes
  • Navigation: All navigation links should work
  • Responsive Design: Dashboard should work on mobile devices

API Endpoints Used

  • /api/club/info - Get club information
  • /api/club/events - Get club events
  • /api/club/members - Get club members
  • /api/admin/clubs/{club_id}/logs - Get club activity logs

Expected Behavior

✅ Success Indicators:
  • Dashboard loads with modern Tailwind CSS design
  • Club name appears in sidebar
  • Statistics cards show real data (members, events, activities)
  • Recent activities table displays club-specific logs
  • Dark mode toggle works and persists
  • Navigation links redirect to appropriate pages
